# 🧸 Will You Go On a Date With Me?

A cute, pink-themed interactive date request page — where saying "No" is basically impossible 💕

## ✨ Features

- Bouncing teddy bear that gets sadder the more you try to say no
- "Nope" button that **runs away** from your cursor every time you hover it
- "Yes" button grows bigger with each escape attempt
- Sweet success screen revealing the date plan 🌸

## 🚀 How to Use

Just open `index.html` in any browser — no setup needed!

Or host it for free with **GitHub Pages**:
1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to **Deploy from branch → main → / (root)**
4. Share the link! 🎉
